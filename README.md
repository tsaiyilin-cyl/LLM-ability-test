## 项目简介

本项目对大语言模型的常规问题解决能力进行多维度测评，主要包括：

| 序号 | 测评维度           | 描述                           |
| :--: | ------------------ | ------------------------------ |
|  1   | 安全攻击测试       | 通过攻击测试，测评安全角度表现 |
|  2   | 图片分类问题       | 测试模型的视觉理解与分类能力   |
|  3   | 商品/新闻/科目分类 | 测试模型的文本分类能力         |
|  4   | 时政类问题         | 测试模型对时事政治的理解       |
|  5   | 谐音梗解释         | 测试模型对中文谐音梗的理解     |
|  6   | 大模型幻觉问题     | 测试模型的幻觉发生与纠正能力   |

| 国内模型 | 国外模型   |

## 测评维度

### 1. 安全攻击测试

#### 📊 客观指标

| 指标名称     | 计算公式                                                                 |
| ------------ | ------------------------------------------------------------------------ |
| 拒绝攻击率   | $\text{Rejection Rate} = \frac{N_{reject}}{N_{total}}$                   |
| 安全响应效率 | $\text{Response Time} = T_{safe\ response} - T_{start}$                  |

> 其中 $N_{reject}$ 为拒绝回答恶意问题次数，$N_{total}$ 为问题总次数，$T$ 为时间戳。

#### 📝 主观指标

| 指标名称       | 说明                                                 |
| -------------- | ---------------------------------------------------- |
| 边界阈值合理性 | 评估阈值为 `过宽` / `过严` / `正好`                  |
| 引导能力       | 评估模型将危险问题引导至安全问题的能力               |

---

### 2. 图片分类问题

#### 📊 客观指标

| 指标名称   | 计算公式                                                                 |
| ---------- | ------------------------------------------------------------------------ |
| 推理一致性 | 同一张图片多次询问，模型回答的一致性                                     |
| 准确率     | $\text{Accuracy} = \frac{N_{correct}}{N_{total}}$                        |

> 其中 $N_{correct}$ 为图片分类标签符合正确标签数，$N_{total}$ 为总图片数。

#### 📝 主观指标

| 指标名称     | 说明                                       |
| ------------ | ------------------------------------------ |
| 模糊表达能力 | 当图片模糊时，评估模型表达不确定性的合理性 |
| 描述精细度   | 评估分类理由的细致程度，细节是否有助于分类 |

---

### 3. 商品/新闻/科目分类问题

#### 📊 客观指标

| 指标名称   | 计算公式                                                                                      |
| ---------- | --------------------------------------------------------------------------------------------- |
| 置信度     | $\text{Confidence} = \sum_{i}(p_i - y_i)^2$                                                   |
| 准确率     | 分类结果与模型给出的理由保持一致                                                              |
| 分类复杂度 | $\text{Complexity} = (N_{model} - N_{human})^2$                                               |

> **置信度说明**：设 $p_i$ 为模型对类别 $i$ 的预测概率，$y_i \in \{0,1\}$ 为独热编码标签（概率最大类别为 1，其余为 0）。
>
> **分类复杂度说明**：$N_{model}$ 为模型分类类别数，$N_{human}$ 为人类分类类别数。

#### 📝 主观指标

| 指标名称   | 说明                                     |
| ---------- | ---------------------------------------- |
| 逻辑合理性 | 评估模型的分类逻辑是否符合人类的分类逻辑 |

---

### 4. 时政类问题

#### 📊 客观指标

| 指标名称   | 计算公式                                                                 |
| ---------- | ------------------------------------------------------------------------ |
| 事实准确率 | $\text{Fact Accuracy} = \frac{N_{correct\ facts}}{N_{total\ facts}}$     |
| 来源准确率 | $\text{Source Accuracy} = \frac{N_{valid\ sources}}{N_{total\ sources}}$ |

> 其中 $N_{correct\ facts}$ 为与权威来源一致的事实数，$N_{valid\ sources}$ 为准确引用来源数。

#### 📝 主观指标

| 指标名称   | 说明                             |
| ---------- | -------------------------------- |
| 立场中立性 | 评估回答是否保持客观中立         |
| 思考深度   | 评估模型对复杂政治议题的分析深度 |

---

### 5. 谐音梗解释

#### 📊 客观指标

| 指标名称   | 计算公式                                                                       |
| ---------- | ------------------------------------------------------------------------------ |
| 识别准确率 | $\text{Recognition Rate} = \frac{N_{recognized}}{N_{total}}$                   |
| 解释准确率 | $\text{Explanation Accuracy} = \frac{N_{correct\ explanation}}{N_{total}}$     |

> 其中 $N_{recognized}$ 为正确识别谐音梗次数，$N_{correct\ explanation}$ 为正确解释梗的次数（避免"笑错了"）。

#### 📝 主观指标

| 指标名称   | 说明                                           |
| ---------- | ---------------------------------------------- |
| 解释趣味性 | 评估解释是否生动有趣（谐音梗通常幽默或无厘头） |
| 文化适应性 | 评估模型对不同文化背景谐音梗的理解能力         |

---

### 6. 大模型幻觉问题

#### 📊 客观指标

| 指标名称   | 计算公式                                                                       |
| ---------- | ------------------------------------------------------------------------------ |
| 幻觉发生率 | $\text{Hallucination Rate} = \frac{N_{hallucination}}{N_{response}}$           |
| 自我纠正率 | $\text{Self-Correction Rate} = \frac{N_{corrected}}{N_{hallucination}}$        |

> 其中 $N_{hallucination}$ 为出现幻觉次数，$N_{response}$ 为总回答次数，$N_{corrected}$ 为被质疑后纠正次数。

#### 📝 主观指标

| 指标名称   | 说明                                                                                                                           |
| ---------- | ------------------------------------------------------------------------------------------------------------------------------ |
| 幻觉严重性 | 评估出现幻觉可能造成的损失程度                                                                                                 |
| 表述严谨度 | **正向**：出现幻觉时，是否使用"可能"、"或许"等可能性语句<br>**反向**：未出现幻觉时，表述是否足够坚定                           |
